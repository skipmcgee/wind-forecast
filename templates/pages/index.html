{% block title %}Analyze Weather Model Accuracy{% endblock %}

There are a variety of wind forecast models (ECMWF, GFS, MBLUE, etc.) which often have different wind forecasts for a given location. How do you determine which model is the most historically accurate at a given location? How do you predict which model will be the most accurate forecast / which you should rely on for your next free flight?

This application uses the free OpenMeteo API to obtain wind forecast data, and a Holfuy device API to obtain the measured wind forecast data. The minimal viable product uses 2 weather forecast models (GFS and ECMWF) and 1 Holfuy device, located at La Bajada Ridge, New Mexico. The codebase can be found at GitHub repository.
